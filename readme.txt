Для компиляции расширений требуется:
1. Установленная Visual Studio 2017 версии 15.9 или позже. Возможна установка бесплатной Community Edition.
2. Установленный .NET Framework 4.7.2 Developer Pack: https://www.microsoft.com/net/download/thank-you/net472-developer-pack
3. Установленный .NET Core SDK 2.2.100 или позже (текущая версия 2.2.100): https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-2.2.100-windows-x64-installer

Для развёртывания веб-сервиса на IIS требуется установить IIS-хостинг версии 2.2.0 или позже (текущая версия 2.2.0):
https://dotnet.microsoft.com/download/thank-you/dotnet-runtime-2.2.0-windows-hosting-bundle-installer
